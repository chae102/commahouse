# 🏡 쉼표 하우스 (Comma House)

## 📖 프로젝트 소개
본 프로젝트는 “지친 일상 속 쉼표”를 제공하는 숙소 예약 웹사이트를 구현하는 것을 목표로 진행되었다.  
사용자는 숙소를 검색하고 상세 정보를 확인한 후 예약까지 진행할 수 있도록 구성하였다.

---

## 🧑‍💻 나의 역할 (팀장)
본 프로젝트에서 팀장으로서 전체 프로젝트를 관리하고 팀원들의 역할을 분담하였다.  
또한 프론트엔드 개발을 중심으로 주요 페이지 구현과 수정 작업을 담당하였다.

- 메인 페이지 설계 및 구현
- 숙소 상세 페이지 최종 수정 및 완성
- 예약 · 결제 페이지 수정 및 기능 보완 참여
- 마이페이지(쿠폰 / 예약내역) 기능 구현
- 전체 페이지 태그 연결 및 이동 구조 설계

👉 프로젝트를 진행하면서 백엔드와의 연동이 이루어지지 않아  
실제 데이터 처리나 기능 구현에 한계를 느꼈다.

---

- 예약 및 결제 페이지 구현  
- 마이페이지(쿠폰, 예약내역) 기능

## 📂 페이지 구성
- 메인 페이지  
- 로그인 / 회원가입  
- 숙소 검색 페이지  
- 숙소 상세 페이지  
- 예약 / 결제 페이지  
- 마이페이지  
- 이벤트 / Q&A / 회사 소개  

---

## 🚀 실행 방법
1. `main1.html` 실행  
2. 상단 메뉴 또는 마이페이지 이동  
3. 로그인 진행  
4. 숙소 검색 (제주로 이동하도록 구현됨)  
5. 숙소 선택 → 상세 페이지 확인  
6. 예약 및 결제 진행 가능  

👉 추가 기능  
- 메인페이지에서 회사 소개, Q&A, 마이페이지 이동 가능  
- 인기 여행지 및 패키지 여행 페이지도 접근 가능  

---

## 💡 문제점 및 한계
- 백엔드 연동이 되지 않아 실제 데이터 저장 및 처리 기능이 제한됨  
- 일부 기능은 UI 중심으로만 구현됨  
- 검색 기능이 특정 지역(제주)으로만 제한됨  

---

## 💭 느낀 점
이번 프로젝트를 통해 웹사이트 제작 과정을 직접 경험하며  
HTML과 CSS를 활용한 UI 구성 능력을 향상시킬 수 있었다.  
또한 팀장으로 프로젝트를 진행하면서 협업의 중요성과  
프로젝트 관리의 어려움을 느낄 수 있었다.
